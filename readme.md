# 🐾 PetPals — Página de Login

## ¿Cómo abrir correctamente?

### ✅ Opción 1 — Doble clic en el script (recomendado)
- **Windows:** Doble clic en `abrir_en_chrome.bat`
- **Mac/Linux:** Doble clic en `abrir_en_chrome.sh` (o ejecutar en terminal: `bash abrir_en_chrome.sh`)

### ✅ Opción 2 — Desde Chrome manualmente
1. Abre Google Chrome
2. Arrastra el archivo `index.html` directamente a la ventana de Chrome
3. ¡Listo!

### ✅ Opción 3 — Desde la barra de Chrome
1. Abre Chrome
2. En la barra de dirección escribe: `Ctrl+L`
3. Arrastra el archivo `index.html` a la barra de URL

---

## ⚠️ ¿Por qué se ve como código?
Si abres el archivo haciendo doble clic desde el explorador de archivos y Chrome lo muestra como texto, 
usa uno de los métodos de arriba. Arrastrar el archivo a Chrome siempre funciona.
